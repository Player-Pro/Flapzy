# Flapzy 🐦

A Flappy Bird-style game built in HTML, CSS, and JavaScript. Dodge floating platforms and see how far you can go!

## How to Play

Open `index.html` in any browser — no installation needed!

| Action | Control |
|--------|---------|
| Flap | `Space` or click |
| Flap (mobile) | Tap anywhere |

## Features

- Golden bird with wing flap animation
- Floating platform obstacles — icy ones from the top, grassy ones from the bottom
- Scrolling sky with parallax clouds
- Score and best score tracked
- Works on both mobile and desktop
- Start screen and game over screen

## How to Run

Just double-click `index.html` or drag it into your browser. That's it!
